# Judging Guide

This file maps AI Social Copilot directly to the hackathon judging rubric.

## Impact 25%

- The project addresses a real and recurring problem: social friction in professional environments.
- It is useful for international students, interns, immigrants, early-career professionals, and multicultural teams.
- It fits `AI for Social Impact` because it improves belonging, confidence, and collaboration rather than only productivity.
- The long-term opportunity includes universities, career centers, workforce programs, and enterprise onboarding.

## Codex App 25%

- The project was built end-to-end with Codex support across product framing, UI implementation, backend integration, debugging, documentation, screenshots, and demo packaging.
- Codex accelerated the path from a rough idea to a functioning prototype and polished submission materials.
- The project naturally splits into parallel-style workstreams:
  - UX and scenario design
  - backend/API integration and structured outputs
  - docs, demo assets, and submission packaging
- If asked about parallel agents or worktrees, present this honestly as the way the build decomposes and the way Codex made those outputs manageable in one environment.

## Creative Use Of Skills 25%

- Coding workflows: frontend implementation, Node backend, API integration, and debugging.
- Developer tasks: schema design, local testing, repository cleanup, screenshot handling, and demo asset packaging.
- Non-technical technical work: README writing, architecture notes, submission copy, demo script writing, and judging-oriented storytelling.

## Demo And Pitch 10%

- The demo is a real app, not a deck or mockup.
- Users can enter inputs, generate live AI guidance, and review a practice flow.
- The repo includes screenshots, architecture notes, and a demo video.
- The product is easy to understand quickly, which helps it stay engaging in a short judging window.

## Feasibility And Scalability 15%

- The current implementation already proves the core user flow.
- The architecture is realistic: browser UI, lightweight Node server, private API key handling, structured LLM outputs.
- Clear expansion paths exist:
  - voice coaching
  - multilingual support
  - saved sessions and personalization
  - organization-specific onboarding
  - analytics for confidence and engagement

## Best Submission Angle

Position AI Social Copilot as:

An inclusion-focused AI product that solves a small but high-frequency workplace pain point, already works as a real app today, and has a believable path to becoming a broader communication support platform.
